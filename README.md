# Greeny-Search
Greeny-Search 是一個基於 Java 與 Spring Boot 開發的智慧搜尋引擎後端系統。它不僅能透過 Google 進行網頁檢索，還能根據使用者定義的關鍵字與權重進行二次排名，並自動過濾與環保相關的內容。🚀 核心功能Google 搜尋整合：透過 GoogleQueryService 模擬瀏覽器行為，抓取 Google 搜尋結果，並自動過濾/解碼 URL。關鍵字加權計分：系統允許使用者輸入多組 Keyword（包含名稱與權重），自動掃描目標網頁內容並計算加權總分。樹狀結構計算 (WebTree)：利用 WebNode 與 WebTree 結構，支援以 Post-order (後序遍歷) 方式計算整個網站群組的階層式得分。自動相關性排序：搜尋結果會優先顯示標題包含關鍵字的網頁，其次依據加權分數進行降序排列。環保議題聚焦：系統內建自動附加關鍵字「環保」，專注於綠色資訊的檢索。📂 專案結構說明Service ：負責處理與 Google 的通訊、Jsoup 解析網頁標籤及計分排序。Controller：提供 RESTful API 接口 /api/search，讓前端可以輕鬆調用搜尋功能。Model 層：WebPage & WordCounter：負責下載網頁內容並統計關鍵字出現次數。WebTree & WebNode：建構網頁間的階層關係，實現複雜的計分演算。Keyword：定義搜尋核心的權重參數。🛠️ 技術棧語言: Java 8+框架: Spring Boot (Web)外部函式庫: Jsoup (HTML 解析)工具: URLConnection, BufferedReader (網路通訊)🚦 如何開始前置準備確保已安裝 JDK 8 或更高版本。準備好 Maven 或相關 IDE (如 Eclipse/IntelliJ)。執行步驟複製專案：Bashgit clone https://github.com/mengchi0202/Greeny-Search.git
cd Greeny-Search
執行後端：啟動 GoogleSearchController 所在的 Spring Boot 應用。API 測試：在瀏覽器輸入：http://localhost:8080/api/search?q=你的關鍵字📊 計分邏輯範例系統會根據以下公式計算分數：$$TotalScore = \sum (KeywordCount \times KeywordWeight)$$並透過 WebTree 的後序遍歷將子節點分數累加至父節點。
